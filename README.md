# Airbnb Project Analysis
## Project Objective
The goal of this project is to analyze the Seattle Airbnb listing dataset and identify the key factors that influence rental pricing and revenue performance. This dashboard helps hosts and investors understand market dynamics, including price sensitivity by location and property size, so businesses can improve listing strategies, target the most profitable segments, and increase annual income.

## Dataset Used
- <a href=https://www.kaggle.com/datasets/alexanderfreberg/airbnb-listings-2016-dataset>Airbnb Dataset </a>
- <a href=https://github.com/vinnythepoo1/Tableau_Project1/blob/main/Airbnb%20project.twb>Airbnb Dataset </a>

## Questions (KPIs)
- How does the average price affect demand?
- Does property size influence average price?
- Which time periods generate the most revenue?
- How does geographic location impact average price?
- What is the property mix by size?
- Is there a price segment for large groups?
